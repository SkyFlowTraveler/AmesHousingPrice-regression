Ames Housing Price Prediction

Описание проекта

Цель проекта — предсказать стоимость домов на основе данных Ames Housing и сравнить различные подходы к моделированию. Основное внимание уделялось работе с категориальными признаками, линейными и бустинговыми моделями, а также исследованию снижения размерности через PCA.

1. Data Cleaning & Preprocessing
2. 
  Проведён анализ категориальных признаков.
  Заполнены пропуски и подобраны кодировщики, которые сохраняют максимальную информативность признаков.

3. Сравнение моделей

Были протестированы следующие модели:

Lasso: RMSE = 34639.6073, R² = 0.8023

Ridge: RMSE = 34423.3494, R² = 0.8045

SVR_linear: RMSE = 74480.8049, R² = 0.1145

SVR_rbf: RMSE = 81111.0758, R² = -0.0492

RandomForest: RMSE = 29289.1763, R² = 0.8599

GradientBoosting: RMSE = 27305.8862, R² = 0.8781

XGBoost: RMSE = 28662.0984, R² = 0.8663

LightGBM: RMSE = 27915.3056, R² = 0.8726

SVR показал себя особенно плохо, что объясняется чувствительностью к масштабу численных признаков.

4. PCA
   
Был проведён эксперимент с PCA на числовых признаках.

Результаты оказались хуже, чем у CatBoost, так как многие важные признаки категориальны.

Кроме того, при использовании PCA не были сгенерированы дополнительные полезные признаки, что снизило качество моделей.


Итоги:

CatBoost — лучший выбор для Ames Housing, так как сразу работает с категориальными признаками и пропусками.

Линейные модели после PCA (Ridge, Lasso) дают базовые результаты, но уступают бустинговым моделям.

Для моделей, чувствительных к масштабу (SVR), важно правильное масштабирование признаков.

Feature importance позволяет выявить ключевые признаки стоимости дома: площадь, качество постройки, количество ванных комнат и т.д.
